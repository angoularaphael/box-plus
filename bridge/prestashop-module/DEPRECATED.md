# Obsolète — utiliser l’intégration JavaScript

Voir : [`../prestashop/README.md`](../prestashop/README.md)

- `bridge/server.js` + `bridge/prestashop-sync.js`

- API Webservice PrestaShop (`PRESTASHOP_URL`, `PRESTASHOP_API_KEY`)
