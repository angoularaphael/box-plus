# Scénario 06 — Séance d'essai gratuite

## Objectif
Workflow sans vente immédiate : fiche prospect, réservation ou note.

## Étapes
1. `m trial-create-member` — créer ou ouvrir fiche prospect
2. `m trial-booking` — si réservation séance
3. OU `m trial-note-only` — note commerciale sans vente
4. `s trial-done`
5. `q`

Valider avec le coach : création vente à 0 € vs note interne uniquement.
